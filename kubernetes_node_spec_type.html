Overview

NodeSpec describes the attributes that a node is created with.

Parameter Summary

  • externalID (Property) -

    External ID of the node assigned by some machine database (e.g. a cloud provider). Deprecated.

  • name (Parameter) -

    Name of the node_spec.

  • podCIDR (Property) -

    PodCIDR represents the pod IP range assigned to the node.

  • providerID (Property) -

    ID of the node assigned by the cloud provider in the format: <ProviderName>://<ProviderSpecificNodeID>

  • unschedulable (Property) -

    Unschedulable controls node schedulability of new pods. By default, node is schedulable. More info: releases.k8s.io/HEAD/docs/admin/node.md#manual-node-administration'`
