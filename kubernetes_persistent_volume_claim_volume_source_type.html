Overview

PersistentVolumeClaimVolumeSource references the user's PVC in the same namespace. This volume finds the bound PV and mounts that volume for the pod. A PersistentVolumeClaimVolumeSource is, essentially, a wrapper around another type of volume that is owned by someone else (the system).

Parameter Summary

  • claimName (Property) -

    ClaimName is the name of a PersistentVolumeClaim in the same namespace as the pod using this volume. More info: releases.k8s.io/HEAD/docs/user-guide/persistent-volumes.md#persistentvolumeclaims

  • name (Parameter) -

    Name of the persistent_volume_claim_volume_source.

  • readOnly (Property) -

    Will force the ReadOnly setting in VolumeMounts. Default false.
