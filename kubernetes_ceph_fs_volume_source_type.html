Overview

CephFSVolumeSource represents a Ceph Filesystem Mount that lasts the lifetime of a pod

Parameter Summary

  • monitors (Property) -

    Required: Monitors is a collection of Ceph monitors More info: releases.k8s.io/HEAD/examples/cephfs/README.md#how-to-use-it

  • name (Parameter) -

    Name of the ceph_fs_volume_source.

  • readOnly (Property) -

    Optional: Defaults to false (read/write). ReadOnly here will force the ReadOnly setting in VolumeMounts. More info: releases.k8s.io/HEAD/examples/cephfs/README.md#how-to-use-it

  • secretFile (Property) -

    Optional: SecretFile is the path to key ring for User, default is /etc/ceph/user.secret More info: releases.k8s.io/HEAD/examples/cephfs/README.md#how-to-use-it

  • secretRef (Property) -

    Optional: SecretRef is reference to the authentication secret for User, default is empty. More info: releases.k8s.io/HEAD/examples/cephfs/README.md#how-to-use-it

  • user (Property) -

    Optional: User is the rados user name, default is admin More info: releases.k8s.io/HEAD/examples/cephfs/README.md#how-to-use-it
