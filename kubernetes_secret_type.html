Puppet Type: kubernetes_secret_type

Defined in:
lib/puppet/type/kubernetes_secret.rb

Overview

Secret holds secret data of a certain type. The total bytes of the values in the Data field must be less than MaxSecretSize bytes.

Parameter Summary

  • data (Property) -

    Data contains the secret data. Each key must be a valid DNS_SUBDOMAIN or leading dot followed by valid DNS_SUBDOMAIN. The serialized form of the secret data is a base64 encoded string, representing the arbitrary (possibly non-string) data value here. Described in tools.ietf.org/html/rfc4648#section-4

  • metadata (Property) -

    Standard object's metadata. More info: releases.k8s.io/HEAD/docs/devel/api-conventions.md#metadata

  • name (Parameter) -

    Name of the secret.

  • type (Property) -

    Used to facilitate programmatic handling of secret data.
