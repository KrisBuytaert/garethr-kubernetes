Overview

ResourceRequirements describes the compute resource requirements.

Parameter Summary

  • limits (Property) -

    Limits describes the maximum amount of compute resources allowed. More info: releases.k8s.io/HEAD/docs/design/resources.md#resource-specifications

  • name (Parameter) -

    Name of the resource_requirements.

  • requests (Property) -

    Requests describes the minimum amount of compute resources required. If Requests is omitted for a container, it defaults to Limits if that is explicitly specified, otherwise to an implementation-defined value. More info: releases.k8s.io/HEAD/docs/design/resources.md#resource-specifications
